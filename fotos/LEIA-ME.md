# 📸 Como adicionar fotos

Esta pasta guarda as **fotos originais** das viagens. Tudo o que estiver aqui vira
quebra-cabeça no app.

## Pelo computador (mais simples para várias fotos)

1. Arraste as fotos para dentro desta pasta `fotos/`.
2. **O nome do arquivo é o nome do lugar.** Ex.: `Cataratas do Iguaçu.jpg`,
   `Praia de Copacabana.jpg`. É esse nome que aparece no app e é lido em voz alta.
3. Pronto. Ao subir para o GitHub, a automação processa e publica sozinha.

## Pelo celular (direto no GitHub)

1. Abra o repositório no GitHub e entre na pasta `fotos`.
2. Toque em **Add file → Upload files** e escolha as fotos do celular.
3. Renomeie cada arquivo com o nome do lugar antes de confirmar, se quiser.
4. Confirme (**Commit changes**). Em alguns minutos as fotos aparecem no app.

## Dicas

- **Formato:** prefira **JPG ou PNG**. O iPhone às vezes salva em HEIC — se der
  problema, ative em *Ajustes → Câmera → Formatos → "Mais Compatível"*, ou
  compartilhe a foto (o iPhone costuma converter para JPG ao compartilhar).
- **Fotos em pé ou deitadas:** não se preocupe, a orientação é corrigida sozinha.
- **Privacidade:** ao processar, a localização (GPS) e outros dados da foto são
  removidos automaticamente.
- **Descrição do lugar:** o texto que é mostrado e lido em voz alta fica em
  `legendas.json` (na mesma pasta). É opcional — sem ele, aparece só o nome.

## Fotos de exemplo

Os arquivos que começam com `exemplo-` são paisagens de teste. Pode apagá-las

# Documentação do Quebra-Cabeças da Vó

Esta pasta registra **tudo** o que foi decidido e construído no projeto, conforme
combinado. A ideia é que qualquer pessoa (ou a Mariana daqui a seis meses) consiga
entender o porquê de cada escolha sem precisar reconstruir o raciocínio.

## Resumo em uma frase

Um **aplicativo web instalável (PWA)** com quebra-cabeças feitos a partir de fotos
de viagens, desenhado para ser usado por uma senhora de 70+ anos, semialfabetizada —
com botões grandes, áudio em tudo e a possibilidade de gravar a própria voz contando
cada lugar.
